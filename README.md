# Portafolio-de-Proyectos
Portafolio de Proyectos de Código

Bienvenido a mi portafolio de proyectos de programación. Aquí podrás encontrar varios proyectos que he desarrollado en diferentes áreas, utilizando varias tecnologías. Cada enlace te redirige a un repositorio individual con el código fuente y documentación detallada de cada proyecto.

## Proyectos

### 1. [Godot Experimentos](https://github.com/Jose-Vicente-Antonio-Gonzalez/Godot.Experimentos.git)
Este repositorio contiene una serie de experimentos desarrollados en Godot, un motor de juego de código abierto. Los proyectos aquí incluyen pruebas y demostraciones de funcionalidades y características del motor, como el manejo de físicas, luces, animaciones y otros aspectos técnicos. Estos proyectos están destinados a explorar y entender el funcionamiento de Godot de manera práctica.

**Tecnologías utilizadas**: Godot Engine.

---

### 2. [Godot Clases Content](https://github.com/Jose-Vicente-Antonio-Gonzalez/Godot.Clases.Content.git)
En este repositorio encontrarás material educativo, utilizado en el desarrollo de las clases como ejercicios y ejemplos relacionado con Godot. Aquí se agrupan diferentes clases y tutoriales que explican las bases y conceptos avanzados del desarrollo de videojuegos usando Godot. Los proyectos incluyen ejemplos de clases prácticas que puedes seguir para aprender a desarrollar juegos y aplicaciones interactivas en este motor.

**Tecnologías utilizadas**: Godot Engine.

---

### 3. [Frontend](https://github.com/Jose-Vicente-Antonio-Gonzalez/Frontend.git)
Este repositorio contiene una colección de proyectos frontend desarrollados con tecnologías web modernas como HTML, CSS, JavaScript, y frameworks como React. Los proyectos en este repositorio muestran la creación de interfaces de usuario (UI) y la implementación de experiencias interactivas en el navegador. Ideal para aprender sobre el desarrollo de aplicaciones web y el diseño de interfaces.

**Tecnologías utilizadas**: HTML, CSS, JavaScript, React.

---

### 4. [General](https://github.com/Jose-Vicente-Antonio-Gonzalez/General.git)
Este repositorio agrupa una variedad de proyectos que no encajan específicamente en las categorías anteriores. Puedes encontrar desde pequeños proyectos personales hasta ejemplos de desarrollo de software en diferentes lenguajes y herramientas. La diversidad de los proyectos refleja diferentes enfoques y prácticas de programación.

**Tecnologías utilizadas**: Python, Javascript, C++, C, Karel, Pascal/Pseint, M. Turing.
